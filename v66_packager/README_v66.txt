Jitsi NDI Portable EXE Launcher Packager v66

Run this inside your working repository:

  cd D:\MEDIA\Desktop\jitsi-ndi-native
  Expand-Archive -Force .\jitsi_ndi_portable_exe_launcher_v66_20260428.zip .
  .\jitsi_ndi_portable_exe_launcher_v66_20260428\make_portable_v66_exe_launcher.ps1

The generated portable ZIP will be placed in the dist folder.
The portable version starts through JitsiNDI.exe.
It launches the PowerShell GUI hidden, without a separate PowerShell console window.
